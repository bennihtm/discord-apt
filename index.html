Discord APT Repo

An unofficial APT repository for the Discord packages updated every 3 hours

The APT repository is built using Github Actions and deployed on Github Pages. The CDN is provided by Cloudflare.

Add the repo

  1. Create a file /etc/apt/sources.list.d/discord.list with the contents deb [signed-by=/usr/share/keyrings/discord-benni.gpg.asc] https://discord-apt.bendaha.eu.org/debian/ ./
  2. Download the file https://discord-apt.bendaha.eu.org/discord-benni.gpg.asc to /usr/share/keyrings/discord-benni.gpg.asc
  3. Run sudo apt update
  4. Install any of the Discord versions: sudo apt install discord / discord-ptb / discord-canary
